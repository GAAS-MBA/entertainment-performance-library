# Entertainment Performance Library

## General Concept

このリポジトリは、ミュージシャン、俳優、プロレスラーなどのエンターテイナーで歴史的にトップのパフォーマンスを出した偉人を**コンピューターとして定義**し、その再現方法について記述する。

### 対象地域

- **USA**（アメリカ合衆国）
- **日本**

### ドキュメント

- [分析候補のスクリーニング基準](docs/SCREENING_CRITERIA.md)
- [エンターテイナー＝コンピューター スペック定義](docs/ENTERTAINER_COMPUTER_SPEC.md)

### アプリケーション

TypeScript + React + Vite で構築したモックアプリ（ライトカラーテーマ、サイドバー付き）。

```bash
npm install
npm run dev
```
