# 分析候補のスクリーニング基準

エンターテイナーを分析対象として選定する際の基準。項目は**多次元**で評価し、各軸を独立してスコア化する。

---

## 選定の前提となる例

スクリーニングフローを決める前提として、以下のような候補を選定対象とする。

| 名前 | ジャンル | 地域 | 備考 |
|------|----------|------|------|
| Michael Jackson | ミュージシャン | USA | |
| The Beatles | ミュージシャン | UK（USA市場で絶大な影響） | |
| Taylor Swift | ミュージシャン | USA | |
| Blackpink | ミュージシャン | 韓国（グローバル市場で絶大な影響） | |
| Johnny Depp | 俳優 | USA | |
| 木村拓哉 | 俳優・タレント | 日本 | |
| アントニオ猪木 | プロレスラー | 日本 | |

**表記ルール**: 日本人以外は origin language（原語）で名前を記載する。

---

## 多次元スクリーニング軸

各候補を以下の軸で評価する。軸は独立しており、総合スコアや個別スコアの組み合わせで判断する。

### 1. 地域・市場軸（Geography / Market）

| 水準 | 説明 |
|------|------|
| 主対象 | USA または 日本に活動の中心がある |
| 拡張 | USA・日本市場で絶大な影響を持つ（他地域出身可） |
| 対象外 | 上記いずれにも該当しない |

### 2. 単位軸（Unit）

| 水準 | 説明 |
|------|------|
| 個人 | 個人としてのパフォーマンスが分析対象 |
| グループ | 集団としてのパフォーマンス・方法論が分析対象 |

### 3. ジャンル軸（Genre）

| 水準 | 説明 |
|------|------|
| ミュージシャン | 歌唱・演奏・ダンスなど |
| 俳優 | 演技・キャラクター表現 |
| プロレスラー | 格闘・エンターテインメント |
| タレント | 司会・バラエティ・複合 |
| その他 | 上記に該当しないエンターテイナー |

※ 複数ジャンルに跨る場合は、主たるジャンルで分類するか、横断的分析対象とする。

### 4. 時代軸（Temporal）

| 水準 | 説明 |
|------|------|
| ピーク明確 | 代表的なパフォーマンス・時期が特定しやすい |
| ピーク分散 | 複数期に渡るが、代表例は挙げられる |
| 継続中 | 現在も活動中でピークが未確定 |

### 5. 市場スケール軸（Scale）

| 水準 | 説明 |
|------|------|
| グローバル | 複数地域でトップクラスの影響 |
| 国内 | 主たる市場（USA または 日本）でトップクラス |
| ニッチ | 特定セグメントで絶対的トップ |

### 6. 記録・観察可能性軸（Observability）

| 水準 | 説明 |
|------|------|
| 高 | 映像・音源・試合記録など一次資料が豊富で、反復観察可能 |
| 中 | 一次資料はあるが限定的、二次資料で補完可能 |
| 低 | 分析に必要な資料がほとんど存在しない |

### 7. 文書化軸（Documentation）

| 水準 | 説明 |
|------|------|
| 高 | インタビュー、自伝、ドキュメンタリーなどが豊富 |
| 中 | 一部の資料が存在 |
| 低 | 方法論を裏付ける資料が乏しい |

### 8. 実績・影響力軸（Achievement / Impact）

| 水準 | 説明 |
|------|------|
| 高 | その分野でトップクラス、後続に明確な影響 |
| 中 | トップクラス、影響は限定的 |
| 低 | 実績・影響が不明確 |

### 9. 独自性・識別性軸（Distinctiveness）

| 水準 | 説明 |
|------|------|
| 高 | 他に類を見ない手法・スタイル、名前だけで想起可能 |
| 中 | 独自性はあるが類似例も存在 |
| 低 | 汎用的で識別性が低い |

### 10. 再現可能性軸（Reproducibility）

| 水準 | 説明 |
|------|------|
| 高 | 方法論・プロセスが観察・記述・抽出可能 |
| 中 | 一部は抽出可能、要検証 |
| 低 | 身体的特徴・時代背景に強く依存し、方法論として抽出困難 |

### 11. 論争・透明性軸（Controversy / Transparency）

| 水準 | 説明 |
|------|------|
| クリア | 実績・貢献について重大な論争なし、客観的評価可能 |
| 論争あり | 一部に論争があるが、パフォーマンス分析は可能 |
| 不透明 | 重大な論争があり、客観的評価が困難 |

---

## スクリーニングフロー

```
候補の列挙
    ↓
各軸でスコアリング（多次元評価）
    ↓
必須水準チェック
  - 地域・市場: 主対象 または 拡張
  - 記録・観察可能性: 低 以外
  - 再現可能性: 低 以外
  - 論争・透明性: 不透明 以外
    ↓ 不適合 → 除外
    ↓ 適合
優先度算出（各軸のスコアを総合）
    ↓
分析候補として採用
```

---

## 優先度の考え方

- 各軸は**独立**して評価し、多次元プロファイルとして可視化可能
- 必須水準を満たし、除外軸に該当しない → 分析対象として採用可能
- 優先度は、高スコアの軸の数や特定軸の重み付けで算出
- 軸間のトレードオフ（例: 文書化は低いが観察可能性は高い）も許容する
